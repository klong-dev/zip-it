# Hướng dẫn tải font Inter

## Cách 1: Tải từ Google Fonts
1. Truy cập: https://fonts.google.com/specimen/Inter
2. Click "Download family"
3. Giải nén file zip
4. Copy các file .ttf hoặc .woff2 vào thư mục này

## Cách 2: Tải trực tiếp (khuyến nghị)
Tải các file sau và đặt vào thư mục này:

### Inter SemiBold (600)
- https://fonts.gstatic.com/s/inter/v13/UcCO3FwrK3iLTeHuS_fvQtMwCp50KnMw2boKoduKmMEVuGKYAZ9hiA.woff2

### Inter Bold (700)
- https://fonts.gstatic.com/s/inter/v13/UcCO3FwrK3iLTeHuS_fvQtMwCp50KnMw2boKoduKmMEVuFuYAZ9hiA.woff2

## Đổi tên file
Sau khi tải, đổi tên các file thành:
- Inter-Regular.woff2
- Inter-Medium.woff2
- Inter-SemiBold.woff2
- Inter-Bold.woff2
